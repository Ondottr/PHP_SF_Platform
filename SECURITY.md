# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |
| dev     | Yes       |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Please report security issues by emailing **dmytro.dyvulskyi@nations-original.com** with the subject line `[SECURITY] PHP Simple Framework`.

Include as much of the following as possible:

- A description of the vulnerability and its potential impact
- Steps to reproduce or a proof-of-concept
- Affected versions
- Any suggested fix, if you have one

You will receive an acknowledgement within **72 hours**. We aim to release a fix within **7 days** for confirmed critical issues.

Once a fix is released we will publicly disclose the issue, crediting you unless you prefer to remain anonymous.
